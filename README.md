#  Hospital Emergency Room Admission Dashboard  
### Power BI • Healthcare Analytics • Emergency Department Insights

This repository contains a fully interactive **Hospital Emergency Room (ER) Admission Dashboard** built using **Power BI**.  
The dashboard delivers deep insights into patient flow, wait times, satisfaction, demographic patterns, and departmental referrals—helping hospitals make informed, data-driven decisions.

---

##  Project Overview

The **Hospital ER Admission Dashboard** is designed to analyze real-world healthcare data, focusing on emergency room operations.  
The aim is to improve patient care, reduce wait times, optimize staffing, and identify important patterns in ER activity.

This project demonstrates strong competencies in:

- Data modeling  
- DAX measures  
- Power Query transformations  
- Visualization design  
- Healthcare KPI analytics  

---

## Key Dashboard Components

### **1. Monthly Summary KPIs**
- Total Patients  
- Average Wait Time  
- Patient Satisfaction Score  
- Number of Referrals  

### **2. Patient Admission Status**
- Admitted vs Not Admitted  
- Percentage distribution  
- KPI indicators  

### **3. Wait Time Target Analysis**
- Patients seen within 30 minutes  
- Patients who exceeded waiting time  
- Donut chart representation  

### **4. Patient Demographics**
Analysis by:
- Age Group  
- Gender  
- Race/Ethnicity  

### **5. Department Referrals**
Breakdown of referrals to:
- General Practice  
- Orthopedics  
- Physiotherapy  
- Cardiology  
- Neurology  
- Gastroenterology  
- Other / None  

### **6. Hourly & Daily Visit Heatmap**
- Hourly patient flow (00:00–23:00)  
- Day-of-week patient frequency (Mon–Sun)  
- Identification of peak hours and busiest days

##  Sample Insights From the Dashboard

- ER traffic peaks around **mid-morning hours**.    
- Patients aged **20–50** represent the majority of ER visits.  
- Referral count is highest in **General Practice** and **Orthopedics**.  
- Only around **40%** of patients are seen within the 30-minute target (example metric).
